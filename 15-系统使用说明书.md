# 15-系统使用说明书

## 1. 登录与项目

1. 启动本地服务后访问 `http://127.0.0.1:5173/`。
2. 注册或登录账号。
3. 在项目列表中新建项目，项目卡片提供“进入工作台、编辑、成员、删除”等操作。
4. 项目 OWNER 在“成员”中输入用户名或邮箱发送邀请；被邀请人必须在自己的项目列表顶部接受邀请后，才会成为项目成员。

## 2. 工作台基础操作

- 顶部“退出项目”按钮用于离开当前工作台，“退出登录”用于结束当前账号会话；如果当前模型存在未保存修改，系统会先询问是否保存。
- 左侧是对象树，可选择、显示/隐藏、锁定/解锁和删除草图实体。
- 中间是三维画布，右键拖动旋转，中键拖动平移，滚轮缩放。
- 右侧是属性面板和在线用户列表，可精确修改选中对象参数。
- 左右侧栏可以拖拽调整宽度，适合在小屏幕或复杂模型中腾出画布空间。
- 选中三维实体时，按住 `Ctrl` 或 `Shift` 点击其他三维实体可以多选；也可以在左侧对象树中使用 `Ctrl/Shift` 多选多个三维特征。

## 3. 草图与平面

- 顶部平面选择支持 `XY`、`XZ`、`YZ`。
- 选择某个平面后，新建线段、矩形、圆、圆弧会创建在该平面。
- 矩形和圆可以在当前草图平面上执行拉伸，拉伸方向为该平面的法向。
- 创建草图实体或三维基本体后，系统会自动回到“选择”工具，便于马上拖动或在右侧编辑参数。

## 4. 工具栏分组

- “草图”：线段、矩形、圆、圆弧。
- “约束”：固定、水平、垂直、尺寸、同心、等半径、平行、垂直关系、相切。
- “实体”：立方体、球体、锥体、拉伸、切除。
- “布尔”：布尔加、布尔减。
- “编辑”：复制、构造切换、旋转、镜像、阵列、偏移、圆角、倒角、删除。
- “视图”：俯视、正视、右视、等轴、全图。

## 5. 约束使用

1. 单对象约束：先选择一个草图对象，再应用固定、水平、垂直或尺寸。
2. 双对象约束：按住 `Ctrl` 或在对象树中多选两个对象，再应用同心、等半径、平行、垂直关系或相切。
3. 相切规则：圆与直线相切时，第二个被选对象会被移动到相切位置；两个圆相切时，第二个圆会移动到外切位置。
4. 右侧属性面板可以查看和删除约束，尺寸约束可以直接修改数值。

## 6. 布尔加与布尔减

- 布尔加：选中两个三维实体后执行，系统会把两个实体合并为一个结果实体。
- 布尔减：选中两个三维实体后执行标准 A-B，系统会从第一个选中的主体实体 A 中减去与第二个选中的客体实体 B 重叠的体积。
- 两个布尔操作都按“第一个选中对象为主体，第二个选中对象为客体/工具体”的顺序创建布尔特征。
- 如果布尔菜单项不可用，通常表示当前没有选中足够的三维实体。

## 7. 草图切除

- 工具栏“实体 -> 切除”用于草图切除。
- 使用时选择主体实体、矩形或圆草图轮廓和切除深度，系统会把草图轮廓沿所在平面的法向拉伸为刀具体，再从主体实体中减去这部分体积。
- 弹窗会优先使用当前选中的三维实体作为主体，并优先使用当前选中的矩形/圆草图作为切除轮廓；也可以在弹窗中手动切换。

## 8. 三维变换、布尔与装配增强

1. 三维实体多选：在视口中按住 `Ctrl` 点击三维实体可增减选择；拖出框选区域时，框内三维实体会优先形成三维多选。
2. 自由移动：拖动立方体、球体、锥体、拉伸体或布尔结果时，位移按当前视角的相机平面计算，可以同时改变 X/Y/Z 坐标，不再只能沿当前草图平面移动。
3. 三维旋转：右侧属性面板可输入 `旋转 X/Y/Z`；工具栏“编辑 -> 三维旋转”可对选中的一个或多个三维实体批量旋转；视口中按住 `Alt` 拖动选中实体可快速旋转。
4. 布尔菜单：只保留“布尔加”和“布尔减”。布尔操作要求显式选中至少两个三维实体，选择顺序决定主体和客体。
5. 装配菜单：选中两个三维实体后，可使用“装配”菜单执行中心重合、X/Y/Z 对齐、Z 面贴合和轴向距离。第一个选中实体作为基准，第二个选中实体作为被约束对象。
6. 固定部件：选中一个或多个三维实体后，使用“装配 -> 固定/解除固定”可锁定或释放部件。被固定部件仍可选中查看，但不能通过视口拖动改变位置。
7. 三平面网格：视口同时显示 XY、XZ、YZ 工作平面网格，当前草图平面更醒目，网格范围会随模型尺寸自动扩展。

## 9. 协同与在线用户

- 进入同一文档的成员会显示在右侧在线用户列表。
- 他人鼠标位置会显示在 3D 视图区的远程光标层。
- 编辑操作会通过 WebSocket 广播为增量更新；如果操作基于旧版本，系统状态栏会显示协同冲突提示。
- 同一个账号重复打开或关闭后重新进入，不会再显示多个重复在线用户。
- 关闭网页、返回项目列表或退出登录时，系统会主动断开当前工作台协同连接。

## 10. 保存与导出

- 点击“保存”会把当前 CAD 文档保存到后端数据库。
- 首页“模型文件 -> 上传模型文件”可以把之前下载的 `.cloudcad` 系统模型文件上传回系统，也兼容旧版 `.json`。每次上传都会自动创建一个独立项目；一个模型文件对应一个项目，删除项目只会删除该项目内的模型文档。上传后系统会自动打开该模型。
- 首页“模型文件 -> 上传模型文件”还支持导入 STL、glTF/GLB 网格模型；STEP/DWG 当前需要先转换为 STL 或 glTF/GLB，后续可接入服务端转换器。
- 首页“模型文件 -> 下载”会把数据库中的 CAD 快照下载为本地 `.cloudcad`，之后可以再次上传回系统继续编辑。
- “版本”可以查看历史保存记录。
- “SVG”和“DXF”用于导出二维草图；“STL”和“GLB”用于导出三维实体模型。

## 11. 常见问题

- 页面打不开：先运行 `scripts\dev\local-status.cmd` 检查数据库、后端、前端端口。
- 拉伸没有效果：确认选中的是非构造的矩形或圆。
- 布尔减没有效果：先选主体 A，再按住 `Ctrl/Shift` 选择客体 B，然后执行“布尔减”；如果 B 与 A 没有空间重叠，外观不会发生明显变化。
- 切除没有效果：确认草图轮廓是非构造的矩形或圆，并且切除深度方向与主体实体有空间重叠。
- 约束没有生效：确认对象数量和类型符合约束规则，例如平行需要两条线段，相切需要圆与线段或两个圆。
- 看不到完整模型：使用“视图 -> 全图”，或中键拖动画布平移视图。
